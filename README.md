# JiYuAuxiliary
一个用易语言写的极域辅助工具
## 软件介绍
顾名思义，这是一个针对极域编写的软件，使用易语言开发 ~~（因为不会写其他语言qwq）~~ 。在静态编译后使用[upx](https://upx.github.io/)加了个压缩壳，将体积和误报毒的可能性在我的能力范围之内降低到到了最小。
## 功能介绍
本程序的功能非常简单，主要功能是解除极域在广播时的全屏状态，可以跟着老师后面边听边做。在信息课上课效率可以进一步提升。

本程序功能如下：  
1.终止/运行极域  
2.挂起/恢复极域  
3.解除U盘和网络锁  
4.解锁广播时的全屏按钮限制  
5.窗口化屏幕广播窗口  
6.最小化屏幕广播和黑屏安静窗口  
7.解除广播或黑屏安静时的键鼠锁  
8.快捷键最小化顶端窗口  
9.反监视（部分系统下不可用）  

其中反监视的原理实际上就是覆盖了一个全屏的置顶窗口，并且设置这个窗口可以穿透并且透明，并调用系统自带的api设置为不可截图。在部分系统中截图呈黑屏状态，但是在Win10 2004中不能用，暂时还没找到解决方法（

另外如果老师已经禁用了U盘或者网络，可以先用命令解除再把本程序丢到机房电脑上

解除网络锁：sc stop tdnetfilter
解除U盘锁：sc stop TDFileFilter

如果打不开cmd的话，可以在任意位置新建快捷方式，将命令打进去，再以管理员身份运行即可（如果是 Administrator 则直接在运行框或者地址栏里面打就行）

### 关于配置文件
本工具支持使用配置文件存储勾选的功能，如果检测到程序运行目录下存在配置文件，则会优先读取配置文件内容，并根据配置文件开启所需功能。配置文件可以在工具内直接生成。

由于本人实力问题，暂时不能修复在Win7等系统下的DPI缩放问题，如果遇到DPI问题，可以下载后缀为Non-Scaling的版本

本程序部分代码使用精易模块11.1.5编写，精易模块下载地址：[https://ec.125.la/](https://ec.125.la/)
